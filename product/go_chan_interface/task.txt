Завдання: Розробка системи обробки рядків

Опис:
Необхідно створити програму, яка буде моделювати обробку даних з використанням горутин, waitGroup, каналів та інтерфейсів. 
Програма повинна читати дані з змінної ListOfLines, обробляти їх у горутинах і передавати результати через канали.  (За додатковий бал можна реалізувати читання з файлу замість ListOfLines, приклад читання та запису в пакеті bufioFiles)

Завдання складається з наступних частин:

Інтерфейс Processor:
Створіть інтерфейс Processor з методом Process(data string) string.
Реалізуйте дві структури, які задовольняють цей інтерфейс:
UppercaseProcessor: Метод Process цієї структури повинен перетворювати вхідний рядок в верхній регістр (всі літери CAPSом).
ReverseProcessor: Метод Process цієї структури повинен повертати вхідний рядок у зворотному порядку (Кіт -> тіК).

Читання даних:
Реалізуйте функцію для читання рядків з файлу (по одному рядку за раз) і передачі їх в канал.

Обробка даних:
Запустіть дві горутини для обробки даних:
Одна горутина повинна використовувати UppercaseProcessor для обробки рядків.
Друга горутина повинна використовувати ReverseProcessor для обробки рядків.
Використовуйте канали для передачі рядків між основною горутиною і горутинами обробки.

Запис результатів:
Реалізуйте функцію для запису оброблених рядків у змінні UppercaseLines та ReverseLines (або в нові файли).
Використовуйте канал для отримання оброблених рядків з горутин обробки.
